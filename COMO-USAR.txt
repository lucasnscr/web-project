==============================================================
  PreventivaDB - Guia rapido para Windows
==============================================================

ARQUIVOS IMPORTANTES

1. instalar.bat
   Use apenas na primeira vez ou quando precisar atualizar o sistema.

2. iniciar.bat
   Use sempre que quiser abrir o sistema.


PASSO 1 - PRIMEIRA INSTALACAO

1. Clique duas vezes em "instalar.bat".
2. Se o Windows pedir confirmacao, escolha para continuar.
3. Aguarde o processo terminar.

O instalador faz tudo sozinho:
- instala Java 21
- instala Node.js com npm
- instala Maven
- instala as dependencias do frontend
- gera a interface web
- cria o arquivo final da aplicacao

Durante a instalacao, a internet e obrigatoria.


PASSO 2 - ABRIR O SISTEMA

1. Clique duas vezes em "iniciar.bat".
2. Aguarde alguns segundos.
3. O navegador vai abrir automaticamente.

Endereco do sistema:
http://localhost:8080


PASSO 3 - FECHAR O SISTEMA

1. Volte para a janela preta que abriu junto com o sistema.
2. Pressione qualquer tecla.

Pronto. O sistema sera encerrado.


SE DER ALGUM PROBLEMA

- Rode novamente o arquivo "instalar.bat".
- Verifique se existe outro programa usando a porta 8080.
- Veja os logs na pasta "logs".

Arquivos de log:
- logs\instalacao.log
- logs\aplicacao.log
